|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.01.  Instalar Visual Studio Code: --------------------------------------------------------------------------------------------------------------------------------|
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Descargar 'Visual Studio Code': https://code.visualstudio.com/download.

            02. Instalar Visual Studio Code.

                    ⦿   En el asistente de instalación dejar marcado:   ☑ 'Agregar PATH…'.                    

            03. Abrir Visual Studio Code.
            
            04. Instalar las siguientes extensiones en Visual Studio Code:
                    
                    ⦿   Auto Close Tag
                    ⦿   Auto Complete Tag
                    ⦿   Auto Rename Tag
                    ⦿   Code Runner
                    ⦿   HTML CSS Support
                    ⦿   IntelliSense for CSS class names in HTML
                    ⦿   JavaScript (ES6) code snippets
                    ⦿   Live Server
                    ⦿   Material Icon Theme
                    ⦿   Path IntelliSense
                    ⦿   Prettier - Code formatter

                    ⦿   Angular Essentials (Version 18)
                    ⦿   Angular Language Service
                    ⦿   Angular Snippets (Version 18)
                    ⦿   TypeScript Importer


|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.02.  Instalar Node Js: -------------------------------------------------------------------------------------------------------------------------------------------|
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Descargar 'node.js': https://nodejs.org/es/.

            02. Instalar node.js.


|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 01.03.  Modificar Directivas en PowerShell: -------------------------------------------------------------------------------------------------------------------------|
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Abrir el PowerShell de Windows
            
            02. Ejecutar el siguiente comando

                    ⦿   PS C:\..\..> Get-ExecutionPolicy
                    ⦿   Restricted

            03. Si aparece 'Restricted', hay que cambiarla a 'Unrestricted' de la siguiente forma:

                    ⦿   PS C:\..\..> Set-ExecutionPolicy -Scope CurrentUser

                          cmdlet Set-ExecutionPolicy en la posición 1 de la canalización de comandos
                          Proporcione valores para los parámetros siguientes:
                          ExecutionPolicy: Unrestricted

            04. Verificar la versión de 'node' y 'npm':

                    ⦿   PS C:\..\..> node --version
                    ⦿   PS C:\..\..> npm --version

                                                                                                                                                      # C = Carpetas
                                                                                                                                                      # A = Archivos

                        frontend/                                       # C. Raíz del Proyecto.
                        ├── .vscode                                     # C. Configuraciones específicas de Visual Studio Code (atajos, depuración, formato).
                        ├── node_modules                                # C. Contiene todas las dependencias instaladas con npm (no se edita manualmente).
                        ├── public                                      # C. Recursos estáticos (imágenes, íconos, manifest.json, etc.).
                        ├── src                                         # C. Carpeta principal del código fuente de Angular.
                        │   ├── app/                                    # C. Carpeta de la aplicación principal (componentes, servicios, módulos).
                        │   │   ├── app.config.server.ts                # A. Configuración para el renderizado del lado del servidor (SSR:Angular Universal).
                        │   │   ├── app.config.ts                       # A. Configuración general de la aplicación (módulos, dependencias, providers).
                        │   │   ├── app.css                             # A. Estilos globales específicos del componente raíz.
                        │   │   ├── app.html                            # A. Plantilla principal 'html' del componente raíz (App).
                        │   │   ├── app.routes.server.ts                # A. Rutas usadas en el lado del servidor (Angular Universal).
                        │   │   ├── app.routes.ts                       # A. Definición de rutas de la aplicación en el navegador.
                        │   │   ├── app.spec.ts                         # A. Archivo de pruebas unitarias para App.
                        │   │   └── app.ts                              # A. Clase principal del componente raíz (App).
                        │   ├── index.html                              # A. Página HTML que carga Angular. Aquí se inserta la app (<app-root></app-root>).
                        │   ├── main.server.ts                          # A. Punto de entrada de Angular Universal para renderizado en servidor.
                        │   ├── main.ts                                 # A. Punto de entrada de la app en el navegador. Inicializa Angular y arranca AppModule.
                        │   ├── styles.css                              # A. Estilos globales de toda la aplicación.
                        │   └── server.ts                               # A. Archivo para configurar el servidor (Node.js + Express).   
                        ├── .editorconfig                               # A. Reglas de estilo para editores de código (indentación, fin de línea, etc.).
                        ├── .gitignore                                  # A. Indica qué archivos/carpetas ignorar en Git (ej: node_modules).
                        ├── angular.json                                # A. Configuración principal Angular (build, serve, rutas de assets, estilos globales).
                        ├── package-lock.json                           # A. Versión exacta de las dependencias instaladas (bloquea versiones).
                        ├── package.json                                # A. Lista de dependencias, scripts (ng serve, ng build, etc.) y metadatos del proyecto.
                        ├── README.md                                   # A. Documentación inicial del proyecto.
                        ├── tsconfig.app.json                           # A. Configuración de TypeScript específica para la aplicación (src/app).
                        ├── tsconfig.json                               # A. Configuración base de TypeScript para todo el proyecto.
                        └── tsconfig.spec.json                          # A. Configuración de TypeScript para pruebas unitarias (Karma/Jasmine).


|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|-- 02.01.  Crear el Proyecto: ------------------------------------------------------------------------------------------------------------------------------------------|
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

            01. Crear la carpeta 'angular' en Windows (Por ejemplo, en el Escritorio):

                    ⦿   C:\..\..\angular>

            02. Abrir con 'Visual Studio Code' la carpeta creada, es decir,'angular'.

            03. Abrir una terminal en Visual Studio Code ('Terminal / New Terminal' ó 'Ctrl + Shift + ñ')
            
            04. Instalar Angular escribiendo desde la terminal:
            
                    ⦿   PS C:\..\..\angular> npm install -g @angular/cli@20.0.1
             
            05. Verificar las versiones de node.js y angular desde la terminal:

                    ⦿   PS C:\..\..\angular> ng version

            06. Crear un proyecto de angular desde la terminal:

                    ⦿   PS C:\..\..\angular> ng new frontend

                          ? Do you want to create a 'zoneless' application without zone.js (Developer Preview)? (y/N) y                          
                          ❯ CSS             [ https://developer.mozilla.org/docs/Web/CSS                     ]
                          ? Do you want to enable Server-Side Rendering (SSR) and Static Site Generation (SSG/Prerendering)? (y/N) y

                ┌ --------------------------------------------------------------------------------------------------------------------------------------- ┐

                  NOTA:   
                    
                    ⦿   Para cambiar el nombre de la terminal, seleccionela en la parte inferior derecha y presionando F2 / Rename...
                    ⦿   Nómbrela: 'frontend'

                    $ git config --global user.name "nombre de su cuenta"

                    $ git init
                    $ git branch -M main
                    $ git remote add origin https://github.com/SenaProfeAlbeiro/angular.git
                    $ git add .
                    $ git commit -m "Subiendo Proyecto"
                    $ git push -u origin main

            05. Va a aparecer una ventana denominada 'Connect to Github', dar clic en la opción 'Sign in with your browser', dar clic en 'Authentication 
                Succeeded'. Verificar que se haya abierto en el navegador que tiene abierta la cuenta, de lo contrario hay que escribir nuevamente las
                credenciales de 'Github'. En el 'Git Bash' debe aparecer texto similar al siguiente:

                    Enumerating objects: 3, done.

                    $ git add .
                    $ git commit -m "Comentario del cambio"
                    $ git push
